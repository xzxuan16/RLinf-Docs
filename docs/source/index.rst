Welcome to **RLInf**!
==============================

RLinf is an efficient, user-friendly, open-source framework for reinforcement learning post-training of large language models (LLMs). 

Our goal is to unleash boundless computing resources, integrate mainstream development frameworks, and streamline the entire development workflow, 
empowering developers to focus on innovation and making RL post-training accessible to everyone!

Key Highlights
--------------

1. **Dynamic Execution System**  
   We introduce a dynamic execution system that adaptively switches between task-collocated and task-separated modes.

2. **Embodied Scenario Support**  
   The first RL training framework designed specifically for embodied environments.

3. **Flexible Backend Integration**  
   Supports multiple training backends and inference engines while maintaining a user-friendly frontend, 
   enabling seamless engine swaps without exposing internal details.
