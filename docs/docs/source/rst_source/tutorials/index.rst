Tutorials
=========

This section helps you get started with **RLInf**.  
It contains a series of hands-on tutorials that walk you through every major aspect of the library.

.. - :doc:`Basic Framework Usage <basic/index>` — a tour of RLInf's static building blocks (configuration files, launching workers, and inter-worker communication).

.. - :doc:`Flexible Execution and Scaling <execution/index>` — the dynamic execution layer: GPU placement, unified scheduling, distributed launch, and fine-grained pipelining.

.. - :doc:`Reinforcement Learning Algorithms <alg/index>` — step-by-step guides for the algorithms currently supported by RLInf (PPO, GRPO, and more).

.. - :doc:`Extending the Framework <extend/index>` — how to add your own algorithms, environments, and models.

- :doc:`Basic Framework Usage <basic/index>` 
   Exploring RLInf's foundational, static components: learn how to define training parameters via YAML configurations, launch and manage worker processes, and establish efficient inter-worker communication.

- :doc:`Flexible Execution and Scaling <execution/index>` 
   Dive into the dynamic execution layer: discover GPU placement strategies, unified resource scheduling, multi-node distributed launch procedures, and fine-grained pipelining optimization.

- :doc:`Reinforcement Learning Algorithms <alg/index>` 
   Following comprehensive hands-on tutorials for each supported RL algorithm, including PPO, GRPO and others, complete with ready-to-use configuration examples and practical performance tips.

- :doc:`Extending the Framework <extend/index>` 
   Learning how to customize RLInf by adding your own training algorithms, integrating new simulation environments, and plugging in new model architectures to meet your research requirements.
